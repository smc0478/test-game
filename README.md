# Square Survival (v0.1)

Minimal browser survival game based on `GAME_SPEC.md`.

## Run Instructions

1. Ensure these files exist in the repository root:
   - `index.html`
   - `style.css`
   - `game.js`
2. Start a local static server:
   - `python3 -m http.server 8000`
3. Open:
   - `http://localhost:8000`
4. Press Arrow keys or WASD to start playing.

## Controls

- Move: Arrow keys + WASD
- Restart: Click `Restart` button on game over

## Spec / Implementation Consistency

- No known mismatches with `GAME_SPEC.md` for v0.1.

# CARD_DESIGN.md (v2.2)

## 카드 철학
- 문양 시너지(Flame/Leaf/Gear/Void)를 빠르게 검증할 수 있도록, 역할 고정형 코어 카드 풀을 우선 제공한다.
- 각 문양은 동일 스펙 카드 4장(공격/방어/드로우/회복)으로 구성해 밸런스 비교 기준을 통일한다.

## v2.2 코어 카드 구조
- 카드 수: 16장(C001~C016)
- 문양 수: 4개, 문양당 4장
- 역할 구성(문양별 동일):
  - 공격: 코스트 1, 피해 8
  - 방어: 코스트 1, 방어 8
  - 드로우: 코스트 1, 드로우 2
  - 회복: 코스트 1, 회복 5

## 시작 덱 규칙
- 시작 덱은 8장으로 고정한다.
- 문양별 공격/방어 카드 2장씩을 넣어 초기 전투에서 모든 문양 시너지를 체험할 수 있게 설계한다.

## 효과/표기 규칙
- 모든 카드는 `id`, `name`, `family`, `type`, `energyCost`, `baseValue`, `sigil`, `effect`, `description`, `image`를 가진다.
- 카드 이미지는 SVG 문자열을 데이터 URL로 생성해 렌더링한다.

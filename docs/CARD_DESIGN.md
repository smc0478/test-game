# CARD_DESIGN.md (v0.7)

## 카드 철학
- 문양 시너지(Flame/Leaf/Gear/Void)를 기반으로 턴 가치가 상승한다.
- v0.7은 **의도 대응 + 임계 조건 + 자원 전환**에 초점을 둔 특이 카드군을 추가했다.

## 신규 카드 의도
- `C027 거울 칼날`: 적 스킬 의도를 역이용해 드로우/에너지 회복.
- `C028 결정 장막`: 공격 의도 적을 카운터하는 방어+가시.
- `C029 역전 톱니`: 방어 자원을 공격으로 뒤집는 전환 카드.
- `C030 영점 폭발`: 마무리 구간(HP 22 이하)에서 폭발 딜.
- `C031 아카이브 스캔`: 카드 도감 후보를 가져와 순간 대응력 강화.

## 효과 설명 표기 규칙
- 모든 카드는 `description` 필드를 가진다.
- UI는 `효과`(기계적 수치) + `설명`(운용 의도)을 동시에 표기한다.

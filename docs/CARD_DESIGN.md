# CARD_DESIGN.md (v1.0)

## 카드 철학
- 문양 시너지(Flame/Leaf/Gear/Void)를 기반으로 턴 가치가 상승한다.
- v1.0은 카드 풀을 100장으로 확장해, **플레이어가 컨셉 덱을 명확히 선택**할 수 있게 하는 것을 목표로 한다.

## v1.0 핵심 설계
- 카드 수: 34장 → 100장(C001~C100)
- 신규 컨셉 카드군:
  - 랜덤 폭발(Chaos/Gamble)
  - 리프 방어-반격(방어 전환)
  - 기어 템포 엔진(드로우+에너지+되감기)
  - 공허 의도 카운터(취약/흡혈/의도 대응)

## 대표 컨셉과 운용 의도
- 랜덤 컨셉: `카이오스 잭팟`, `난수 도박` 계열로 턴 고점을 노린다.
- 리프 방어 컨셉: 방어도를 쌓은 뒤 `방어 전환`으로 공격으로 바꿔 역전한다.
- 기어 템포 컨셉: 드로우/되감기 중심으로 운용하되, 드로우+에너지 동시 수급은 금지해 무한 루프를 방지한다.
- 공허 카운터 컨셉: 적 의도 조건 카드를 활용해 효율적으로 반격한다.

## 효과 설명 표기 규칙
- 모든 카드는 `description` 필드를 가진다.
- UI는 `효과`(기계적 수치) + `설명`(운용 의도)을 동시에 표기한다.


## v1.3.1 밸런스 메모
- 기어 카드군은 전체적으로 수치가 하향되어, 단일 턴 과폭발보다 안정적인 순환 운영을 목표로 한다.
- 드로우 카드에서 에너지 충전을 제거해 시너지 폭주를 막고, 카드 선택의 기회비용을 분명히 했다.

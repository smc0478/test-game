# Synergy Turn Card Battle v0.1 카드 설계

## 카드 종류 설명
v0.1 카드 풀은 총 10장으로 고정되며, 두 가지 타입으로 구성됩니다.

- **공격(attack) 5장**: 직접 피해를 담당
- **스킬(skill) 5장**: 방어/드로우/버프/디버프 담당

문양은 `Flame`, `Leaf`, `Gear`, `Void` 4종이며, 턴 내 사용 누적으로 시너지와 점수 배수를 유도합니다.

## 각 카드 역할
### 공격 카드
- **C001 Ember Strike (Flame, 7)**: 기본 화염 공격
- **C002 Ember Strike+ (Flame, 9)**: 상위 화염 공격
- **C003 Thorn Jab (Leaf, 6)**: 안정적인 중간 피해
- **C004 Cog Shot (Gear, 6)**: 기어 시너지 연계를 위한 공격 축
- **C005 Null Pierce (Void, 5)**: 피해량은 낮지만 점수 배수 시너지 핵심

### 스킬 카드
- **C006 Bark Guard (Leaf, 8 block)**: 기본 방어 확보
- **C007 Clockwork Guard (Gear, 7 block)**: 기어 문양 수급 + 방어
- **C008 Spark Cycle (Gear, draw 1)**: 손패 순환 엔진
- **C009 Ashen Focus (Flame, next attack +2)**: 다음 공격 강화
- **C010 Void Echo (Void, reduce block 2)**: 적 방어 해제 및 Void 연계

## 시너지 설계 의도
시너지 설계는 “같은 문양 집중”과 “다문양 분산”의 선택지를 동시에 제공합니다.

- **집중 보상(문양 2회)**:
  - Flame: 피해 상승으로 즉시 킬 압박
  - Leaf: 방어 효율 상승으로 생존 안정화
  - Gear: 추가 드로우로 연쇄 플레이 확률 증가
  - Void: 카드 단위 점수 효율 상승
- **분산 보상(4문양 완성)**:
  - 턴 점수 x2 + 턴당 1회 +20 보너스
  - 플레이 순서 최적화의 전략적 가치 강화

## 밸런스 방향성
v0.1 밸런스는 복잡한 메타보다 검증 가능한 수치 안정성을 우선합니다.

- 카드 에너지 비용을 전부 1로 통일해 턴 경제를 단순화
- 공격/스킬 점수 가중치(10/8)로 역할 차등 부여
- HP/Block 규칙을 명시적으로 고정해 QA 검증 용이성 확보
- 동시 KO 패배 규칙으로 종료 판정 일관성 확보

## 향후 확장 전략
다음 버전 확장은 반드시 스펙 버전 업데이트와 함께 진행합니다.

- 카드 수 증가(10 -> N)와 희귀도/역할군 분화
- 문양 간 교차 시너지(예: Flame+Gear) 추가
- 적 전용 카드/행동 패턴 도입
- 점수 시스템 확장(콤보/무피해 보너스 등)
- 카드 획득/제거/강화를 포함한 덱 메타 루프 도입

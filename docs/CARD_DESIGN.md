# CARD_DESIGN.md (v0.9)

## 카드 철학
- 문양 시너지(Flame/Leaf/Gear/Void)를 기반으로 턴 가치가 상승한다.
- v0.9는 **단일 문양 빌드만으로도 최종 라운드 진입이 가능하도록** 문양별 보정 장치를 강화했다.

## v0.9 핵심 설계
- 2스택 시너지의 기초 수치를 상향해 단일 문양의 기본 효율을 확보.
- 3스택 달성 시 문양 버스트를 발동해 각 문양의 승리 플랜을 명확히 분리.
  - Flame: 순간 화력
  - Leaf: 생존/회복
  - Gear: 템포/자원 확장
  - Void: 압박 + 역전

## 밸런스 조정 카드 의도
- `C006 바크 가드`: Leaf 단일 운영의 초반 안정성 개선.
- `C009 애쉬 포커스`: Flame 피니시 턴의 폭발력 강화.
- `C013 숲의 의식`: Leaf 운영 시 긴 라운드 생존력 보강.
- `C015 기어 오버클럭`: Gear 운영의 손패 순환/에너지 전개 동시 확보.
- `C022 연쇄 톱니`: Gear 반복 플레이 보상 강화.
- `C026 공허 역류`: Void 의도 읽기 성공 시 리턴 강화.
- `C033 카이오스 잭팟`: 무작위 리스크를 줄이고 시너지 축과 결합.

## 효과 설명 표기 규칙
- 모든 카드는 `description` 필드를 가진다.
- UI는 `효과`(기계적 수치) + `설명`(운용 의도)을 동시에 표기한다.

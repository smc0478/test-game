# Project Changelog

## v1.4.1
### Changed
- 적 턴 카드 사용 로직을 행동 슬롯 제한 방식에서 에너지 소진 우선 방식으로 변경.
- 적이 플레이 가능한 카드가 있는 동안 연속으로 카드를 사용해 남은 에너지를 최대한 소모하도록 수정.
- 무한 반복 방지를 위해 적 턴 행동 안전 상한(12회)을 추가.

## v1.4.0
### Added
- 캔버스 기반 전장 렌더러를 추가해 VS 스테이지를 실시간 드로잉으로 전환.
- 턴 전환 시 플레이어/적 공격 펄스 애니메이션을 추가.

### Changed
- 턴 종료 버튼을 상단에서 하단 전투 조작 섹션으로 재배치해 카드 선택과 조작 동선을 정렬.
- 전투 안내 문구를 상태 기반으로 갱신해 현재 진행 단계(플레이어 턴/적 턴)를 명확화.

## v1.3.1
### Changed
- 기어 카드 전반의 수치를 하향 조정해 템포 엔진 고점을 낮춤.
- 드로우 카드와 에너지 충전이 동시에 붙는 기어 조합을 제거해 무한 콤보 가능성을 차단.
- 확장 카드의 `기어 순환`, `시간 메아리` 효율을 하향 조정.

## v1.3.0
### Added
- 라운드 기반 적 위협 단계(1~4)와 에너지/드로우 보너스를 추가.
- 적 턴 멀티 행동(최대 3회 카드 사용) 규칙을 추가.
- 배틀 스테이지(VS 패널), HP 바, 턴 기반 간단 공격 애니메이션을 추가.

### Changed
- 적 턴이 단일 카드 사용에서 다중 카드 연속 사용으로 변경.
- 난이도 안내 문구를 전투 로그 및 UI에 추가.

## v1.2.0
### Added
- 전투 화면에 문양별 시너지 효과/버스트 규칙 안내 패널을 추가.
- localStorage 기반 자동 저장, `이어하기`, `저장 초기화` 기능을 추가.
- `hall.html` 명예의 전당 페이지를 추가해 승/패, 최고 점수, 격파한 최종 적, 주요 카드 기록을 제공.

### Changed
- 메인 UI 헤더에 명예의 전당 이동 버튼을 추가.
- 버전 표기를 v1.2로 갱신.

## v1.1.0

# Project Changelog

## v0.5.1
### Changed
- PLAYBOOK에 루프 분기 기준을 명시해 버그 수정 루프와 스펙 변경 루프(버전 업) 전환 조건을 구체화했습니다.
- 운영 주기 규칙을 추가해 배포 사이클마다 루프 판단을 반복 수행하도록 정리했습니다.

### Fixed
- 전투 해상도(`resolution`)에서 양측 생존 시 다음 플레이어 턴으로 복귀하지 않던 턴 잠김 버그를 수정했습니다.

## v0.5.0
### Added
- 카드에 `family` 필드를 추가해 동명 계열 트리거를 지원했습니다.
- 신규 카드 4장(C021~C024)을 추가했습니다.
- 자가 피해형 카드(`혈화 강타`)를 추가했습니다.
- 전 턴 사용 기억 기반 조건부 카드 효과(`ifLastTurnFamily`)를 추가했습니다.
- 같은 턴 동명 카드 공명(`echoAttack`) 효과를 추가했습니다.

### Changed
- 카드 풀을 20장에서 24장으로 확장했습니다.
- 적 덱/체력을 신규 카드 메타 기준으로 조정했습니다.
- 카드 UI에 패밀리 정보와 한글 효과 설명을 추가했습니다.

### Fixed
- 취약(`vulnerable`) 상태가 턴 시작마다 1 감소하도록 수정했습니다.

## v0.4.0
### Added
- 지역/라운드 기반 런 구조(3지역, 6라운드)를 추가했습니다.
- 전투 승리 후 카드 선택 덱 빌딩(`deckBuild`) 상태를 추가했습니다.
- 적 타입을 6종으로 확장했습니다.
- 카드 풀을 20장으로 확장하고 카드별 고유 효과를 추가했습니다.
- `runComplete` 상태와 런 클리어 목표를 추가했습니다.

### Changed
- UI를 런 정보(지역/라운드/목표) 중심으로 재구성했습니다.
- 카드 아트를 전 카드에 이미지(data URI SVG)로 적용했습니다.

### Fixed
- 라운드 전환 시 플레이어 체력 연동 및 턴 정보 표시를 안정화했습니다.

## v0.3.0
- Combo Chain / Prism Burst / Adrenaline 시스템 추가.

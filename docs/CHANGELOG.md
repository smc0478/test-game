# Project Changelog

## v0.3.0
### Added
- **Combo Chain** 시스템을 추가했습니다. (연속으로 다른 문양 사용 시 체인 증가, 동일 문양 반복 시 리셋)
- 체인 4 도달 시 턴당 1회 발동하는 **Prism Burst**(피해 8 + 방어 8 + 점수 보너스)를 추가했습니다.
- 한 턴에 서로 다른 문양 시너지 2종 이상 활성화 시 발동하는 **Adrenaline**(+1 에너지, 1드로우, 점수 보너스)를 추가했습니다.

### Changed
- 전투 기본 수치를 v0.3 기준으로 재조정했습니다. (플레이어 HP 64 / 적 HP 78)
- 풀 스펙트럼 보너스를 +30으로 상향하고, 승리 보너스를 +140(+고HP +50)으로 조정했습니다.
- 카드 밸런스를 조정했습니다. (`C002`, `C005`, `C008`, `C009`, `C010`)
- 점수 계산에 콤보 보너스(체인 3:+4, 체인 4:+8)를 반영했습니다.

### Fixed
- 적 AI의 피해 예측이 Flame 시너지 잠재값을 반영하지 못하던 문제를 수정했습니다.

## v0.2.0
### Added
- 카드 2종(`C011 Verdant Pulse`, `C012 Abyss Cut`)을 추가해 카드 풀/덱을 12장 체계로 확장했습니다.
- 카드 오브젝트에 `image` 필드를 도입하고, 손패 렌더링에 카드 아트를 표시하도록 구현했습니다.
- 1턴 내 3번째 카드 사용 시 에너지 +1을 얻는 **Momentum** 규칙을 추가했습니다.
- 손패 제한(8장) 초과 드로우 시 카드를 버린 더미로 보내는 소각 규칙을 추가했습니다.

### Changed
- 점수 시스템을 플레이어 카드 해소 기준으로 정렬하여, 적 카드 해소 시 점수가 증가하지 않도록 변경했습니다.
- 보너스 점수를 v0.2 기준으로 조정했습니다. (승리 +120, 고HP +40, 풀 스펙트럼 +25)
- UI 텍스트를 플레이어 표시 기준 한국어로 통일했습니다.

### Fixed
- 초기 화면 HP 표기를 스펙 값(플레이어 60 / 적 70)과 일치시켰습니다.

## v0.1.0
### Added
- Synergy Turn Card Battle 기준의 수치형 전투 스펙을 정의했습니다.
- 상태 머신(`ready`, `playerTurn`, `enemyTurn`, `resolution`, `gameOver`)과 전이 규칙을 명시했습니다.
- 10장 고정 카드 풀, 문양 시너지, 점수 배수 규칙을 문서화했습니다.

# Project Changelog

## v0.4.0
### Added
- 지역/라운드 기반 런 구조(3지역, 6라운드)를 추가했습니다.
- 전투 승리 후 카드 선택 덱 빌딩(`deckBuild`) 상태를 추가했습니다.
- 적 타입을 6종으로 확장했습니다.
- 카드 풀을 20장으로 확장하고 카드별 고유 효과를 추가했습니다.
- `runComplete` 상태와 런 클리어 목표를 추가했습니다.

### Changed
- UI를 런 정보(지역/라운드/목표) 중심으로 재구성했습니다.
- 카드 아트를 전 카드에 이미지(data URI SVG)로 적용했습니다.

### Fixed
- 라운드 전환 시 플레이어 체력 연동 및 턴 정보 표시를 안정화했습니다.

## v0.3.0

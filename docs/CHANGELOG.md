# Project Changelog

## v1.3.0
### Added
- 라운드 기반 적 위협 단계(1~4)와 에너지/드로우 보너스를 추가.
- 적 턴 멀티 행동(최대 3회 카드 사용) 규칙을 추가.
- 배틀 스테이지(VS 패널), HP 바, 턴 기반 간단 공격 애니메이션을 추가.

### Changed
- 적 턴이 단일 카드 사용에서 다중 카드 연속 사용으로 변경.
- 난이도 안내 문구를 전투 로그 및 UI에 추가.

## v1.2.0
### Added
- 전투 화면에 문양별 시너지 효과/버스트 규칙 안내 패널을 추가.
- localStorage 기반 자동 저장, `이어하기`, `저장 초기화` 기능을 추가.
- `hall.html` 명예의 전당 페이지를 추가해 승/패, 최고 점수, 격파한 최종 적, 주요 카드 기록을 제공.

### Changed
- 메인 UI 헤더에 명예의 전당 이동 버튼을 추가.
- 버전 표기를 v1.2로 갱신.

## v1.1.0
### Added
- 분기 선택 전용 상태 `routeSelect` 및 경로 선택 UI를 추가.
- 경로 모디파이어 4종(선제 압박/보급 지점/요새 진입/지형 우세)을 추가.
- 지역 6개, 적 14종으로 콘텐츠를 확장.
- 도감에 지역 목록 섹션을 추가.

### Changed
- 런 길이를 10라운드로 확장.
- 전투 종료 후 흐름을 `deckBuild -> routeSelect -> next round`로 변경.

## v1.0.0
### Added
- 카드 풀을 100장(C001~C100)으로 확장하고 컨셉 카드군(랜덤/리프 반격/기어 템포/공허 카운터)을 추가.
- `codex.html`에 적 도감(컨셉/행동 패턴/카운터 팁) 섹션을 추가.
- `codex.html`에 추천 덱 운영 가이드(핵심 카드/운용법) 섹션을 추가.

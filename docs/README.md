# Synergy Turn Card Battle v0.1 문서

## 게임 소개
Synergy Turn Card Battle은 **턴제 카드 전투**를 핵심으로 하는 브라우저 게임입니다. 플레이어와 AI 적이 번갈아 턴을 진행하며, 카드의 수치 효과(공격/방어/드로우/버프/디버프)와 문양(Sigil) 시너지 조합으로 전투와 점수를 만들어 갑니다.

v0.1은 Vertical Slice 단계로, 단일 전투 1회를 안정적으로 완료하는 데 집중합니다.

## 핵심 시스템 요약
- **상태 머신 기반 전투 흐름**: `ready -> playerTurn -> enemyTurn -> resolution -> gameOver`
- **에너지 경제**: 턴당 에너지 3, 카드마다 고정 에너지 비용
- **카드 존 구조**: 드로우/손패/버림더미, 드로우 고갈 시 버림더미 셔플
- **문양 시너지**:
  - Flame 2장 이상: 공격 카드 피해 +3
  - Leaf 2장 이상: 방어 카드 방어도 +3
  - Gear 2장 이상: 카드 해결 시 추가 드로우 +1
  - Void 2장 이상: 해당 카드 점수 x1.5(내림)
- **글로벌 턴 배수**: 한 턴에 4문양(Flame/Leaf/Gear/Void) 모두 1장 이상 사용 시 그 턴 남은 카드 점수 x2, 그리고 턴당 1회 +20 보너스
- **점수 시스템**: 공격 카드 +10, 스킬 카드 +8, 승리 +100, 승리 시 HP 40 이상 +30, 패배 시 최종 점수 0

## 플레이 방법
1. `Start Battle` 버튼으로 전투를 시작합니다.
2. 플레이어 턴에서 손패의 `Play` 버튼으로 카드를 사용합니다.
3. 에너지가 0이 되거나 더 낼 카드가 없으면 자동으로 턴이 종료되며, 필요하면 `End Turn` 버튼으로 수동 종료할 수 있습니다.
4. 적 턴은 AI 우선순위 규칙에 따라 자동 진행됩니다.
5. 전투 종료 시 로그와 최종 점수를 확인합니다.

## 승리 조건
- **승리**: 적 HP가 0 이하
- **패배**: 플레이어 HP가 0 이하
- **동시 KO**: v0.1 안전 규칙에 따라 패배 처리

## 기술 스택
- HTML5 (`index.html`)
- CSS (`style.css`)
- Vanilla JavaScript (`game.js`)
- 로컬 실행용 정적 서버(예: `python3 -m http.server`)

## 실행 방법
1. 프로젝트 루트에서 아래 명령 실행:
   - `python3 -m http.server 8000`
2. 브라우저에서 아래 주소 접속:
   - `http://localhost:8000`
3. `Start Battle` 클릭 후 플레이
